# Session Log — POLIGONNE

Log de todas as sessoes de desenvolvimento. Cada chat do Claude Code **DEVE** atualizar este arquivo ao final do trabalho.

---

## Como usar

1. **Ao iniciar um chat novo** → ler este arquivo para saber o que ja foi feito
2. **Durante o trabalho** → registrar decisoes importantes em tempo real (opcional)
3. **Ao encerrar ou quando o contexto estiver enchendo** → adicionar entrada com o resumo

## Formato de entrada

```markdown
## [DATA] — [Tema principal]
**Chat:** [identificador curto, ex: "Feiticos", "Monstros", "Bug fix combate"]
**Arquivos modificados:** lista dos arquivos tocados
**O que foi feito:**
- item 1
- item 2
**Decisoes tomadas:**
- decisao importante que afeta outros sistemas
**Pendente / proximos passos:**
- o que ficou pra fazer
```
