## 🎨 DICAS DE DESIGN

**Temática:**
- Mantenha coerência: Ratos → agilidade/força, Pássaros → esquiva/sorte
- Nomes evocativos: "Rato Alfa" não "Rato Grande"
- Cores e ícones temáticos para fácil identificação

**Balanceamento:**
- Use `mostrarPontosItem()` para validar todos os itens
- Boss (monstro 4) deve ter HP 2-3× maior
- Recursos raros devem ser realmente raros (não essenciais para todo item)

**Progressão:**
- Faixa de 5-10 níveis por família
- Monstro 1: Introdução (fácil)
- Monstro 2-3: Desafio moderado
- Monstro 4: Boss (hard)
