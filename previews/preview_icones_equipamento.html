Preview: Ícones Padrão de Equipamentos

Cada slot de equipamento terá um ícone SVG base que varia em cor automaticamente.
Biblioteca: game-icons.net (CC BY 3.0) - SVGs recoloráveis via CSS

1. Opções de Ícone por Slot

Cada slot mostra variantes do ícone base em diferentes cores. Escolha o ícone padrão para cada tipo.

Capacete / Chapéu (helmet)
Slot: helmet - Proteção de cabeça
Opção A: Elmo com Viseira (visored-helm, by Lorc)
Opção B: Chapéu Pontudo / Mago (pointy-hat, by Lorc)
Opção C: Fedora (fedora, by Lorc)
Peitoral / Colete (chest)
Slot: chest - Armadura do torso
Opção A: Peitoral (breastplate, by Lorc)
Opção B: Colete de Couro (leather-vest, by Lorc)
Calças / Grevas (legs)
Slot: legs - Proteção de pernas
Opção A: Grevas (greaves, by Delapouite)
Botas (boots)
Slot: boots - Calçados
Opção A: Botas (boots, by Lorc)
Capa (cape)
Slot: cape - Capas e mantos
Opção A: Capa (cape, by Delapouite)
Cinto (belt)
Slot: belt - Cintos e faixas
Opção A: Cinto (belt, by Lucasms)
Anel (ring)
Slot: ring0-ring9 - Anéis mágicos
Opção A: Anel Simples (ring, by Delapouite)
Opção B: Anel com Diamante (diamond-ring, by Delapouite)
Opção C: Anel de Poder (power-ring, by Delapouite)
Arma (weapon)
Slot: weapon / weaponLeft - Armas já usam ícones próprios, mas teriam um fallback
Ícone genérico caso arma não tenha ícone definido

2. Simulação: Como ficaria no Inventário

Ícones em tamanho real dentro de slots de inventário (fundo escuro)
